# 🚀 快速启动指南

## 端口配置

- **API 后端**: http://localhost:9001
- **Web 前端**: http://localhost:9002

## 一键启动

### 方式1: 分别启动（推荐）

**终端1 - 启动 API 后端**:
```bash
cd apps/api
pnpm dev
```
✅ API 将在 http://localhost:9001 运行

**终端2 - 启动 Web 前端**:
```bash
cd apps/web
pnpm dev
```
✅ Web 将在 http://localhost:9002 运行

### 方式2: 使用并发启动

```bash
# 在项目根目录
cd apps/api && pnpm dev & cd ../web && pnpm dev
```

## 访问 DeGift 功能

启动完成后，访问以下页面：

| 功能 | URL |
|------|-----|
| 🎁 创建礼物 | http://localhost:9002/gift/create |
| 📋 礼物列表 | http://localhost:9002/gifts |
| 🔍 礼物详情 | http://localhost:9002/gift/[id] |

## ✅ 验证服务

```bash
# 检查 API 是否运行
curl http://localhost:9001

# 检查 Web 是否运行
open http://localhost:9002
```

## 🎯 快速测试

1. 打开浏览器访问 http://localhost:9002/gift/create
2. 连接钱包
3. 选择礼物类型（Token 或 NFT）
4. 填写表单并创建礼物
